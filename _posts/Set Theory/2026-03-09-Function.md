## Definition 1
Let $X, Y$ be sets. A ***function*** from $X$ to $Y$ is a relation $f$ from $X$ to $Y$ satisfying 
**(a)** $\operatorname{Dom}(f) = X$,
**(b)** If $(x, y) \in f$ and $(x, z) \in f$, then $y = z$.

## Definition 2
Let $f : X \to Y$ be a function.

**(i)** $f$ is said to be ***injective*** or ***one-to-one*** if $x_1, x_2 \in X$ with $f(x_1) = f(x_2)$, then $x_1 = x_2$. 

**(ii)** $f$ is said to be ***surjective*** or ***onto*** if $y \in Y$, then there exists $x \in X$ such that $f(x) = y$. In other words, $f : X \longrightarrow Y$ is surjective $\iff f(X) = Y$.

**(iii)** $f$ is said to be ***bijective*** if it is both injective and surjective. A bijection is also called a ***one-to-one correspondence***.

## Theorem 1
Let $f, g : X \longrightarrow Y$ be functions. Then $f = g \iff f(x) = g(x), \forall x \in X$. 

### Proof
$\blacksquare$

## Definition 3
Let $f : X \longrightarrow Y$ be a function, and let $A \subseteq X$ and $B \subseteq Y$, respectively. Then

**(i)** The ***image*** of $A$ under $f$, which we denote $f(A)$, is the set 

$$f(A) = \{ f(x) \mid x \in A \}.$$

**(ii)** The ***inverse image*** of $B$ under $f$, which we denote $f^{-1}(B)$, is the set 

$$f^{-1}(B) = \{ x \mid f(x) \in B \}.$$

## Theorem 2
Let $f : X \longrightarrow Y$ be a function. Then

**(i)** $f(\emptyset) = \emptyset$.

**(ii)** $f(\{x\}) = \{ f(x) \}, \forall x \in X$.

**(iii)** $A \subseteq B \subseteq X \Longrightarrow f(A) \subseteq f(B)$.

**(iv)** $C \subseteq D \subseteq Y \Longrightarrow f^{-1}(C) \subseteq f^{-1}(D)$.

### Proof
$\blacksquare$

## Theorem 3
Let $f : X \longrightarrow Y$ be a function. Then 

**(i)** $f \left(\bigcup_{\gamma \in \Gamma} A_{\gamma}\right) = \bigcup_{\gamma \in \Gamma} f(A_{\gamma})$.

**(ii)** $f \left(\bigcap_{\gamma \in \Gamma} A_{\gamma}\right) \subseteq \bigcap_{\gamma \in \Gamma} f(A_{\gamma})$.

### Proof
$\blacksquare$
