## Theorem
Let $f : X \longrightarrow Y$ be a function. Then

**(i)** If there exists a function $g : Y \longrightarrow X$ such that $g \circ f = id_X$, then $f : X \longrightarrow Y$ is injective.

**(ii)** If there exists a function $h : Y \longrightarrow X$ such that $f \circ h = id_Y$, then $f : X \longrightarrow Y$ is surjective.

## Definition
Let $f : X \longrightarrow Y$ be a function.

**(i)** The ***identity function*** on $X$ is a function $id_X : X \longrightarrow X$ defined by $id_X(x) = x, \forall x \in X$.

**(ii)** The ***inverse function*** of $f$ is a function $g : Y \longrightarrow X$ defined by $g \circ f = id_X$ and $f \circ g = id_Y$. We denote $g = f^{-1}$.

## Remark
A function $f: A \longrightarrow B$ is bijective $\iff$ $f$ has the inverse.
