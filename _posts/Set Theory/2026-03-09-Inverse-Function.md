## Definition
Let $X$ and $Y$ be sets, and let $f : X \longrightarrow Y$ be a function.

(i) The identity function on $X$ is a function $id_X : X \longrightarrow X$ defined by $id_X(x) = x, \forall x \in X$.

(ii) The inverse function of $f$ is a function $f^{-1} : Y \longrightarrow X$ defined by $f^{-1} \circ f = id_X$ and $f \circ f^{-1} = id_Y$.
