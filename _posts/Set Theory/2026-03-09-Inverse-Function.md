## Theorem
Let $f : X \to Y$ be a function. Then

**(i)** There exists a function $g : Y \to X$ such that $g \circ f = id_X$ $\iff$ $f : X \to Y$ is injective.

**(ii)** There exists a function $h : Y \to X$ such that $f \circ h = id_Y$ $\iff$ $f : X \to Y$ is surjective.

### Proof
$\blacksquare$

## Definition
Let $f : X \to Y$ be a function.

**(i)** The ***identity function*** on $X$ is a function $id_X : X \to X$ defined by $id_X(x) = x, \forall x \in X$.

**(ii)** The ***inverse function*** of $f$ is a function $g : Y \to X$ defined by $g \circ f = id_X$ and $f \circ g = id_Y$. We denote $g = f^{-1}$.

## Remark
A function $f: A \to B$ is bijective $\iff$ $f$ has the inverse.
