## Theorem
Let $E$ be open in $\mathbb{R}^n$, and let $\mathbf{f}: E \to \mathbb{R}$ be differentiable on A. If A contains the line segment with end points a and a +h, then there is a point c = a+toh with 0 < to < 1 of this line segment such that f(a+h) - f(a) = Df(c).h.
