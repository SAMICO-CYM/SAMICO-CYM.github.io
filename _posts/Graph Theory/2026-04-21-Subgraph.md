## Definition
**(i)** A graph $H$ is a ***subgraph*** of a graph $G$ if $V(H) \subset V(G)$ and $E(H) \subset E(G)$.

**(ii)** A subgraph $H$ of $G$ is called an ***induced subgraph*** if $E(H) = \\{ (u,v) \in E(G) \mid u,v \in V(H) \\}$. For a subset $A \subset V(G)$, the induced subgraph of $G$ with the vertex set $A$ is denoted by $G \vert_A$.

예를 들어 $P_3$을 생각하자. 그러면 $V(P_3) = \\{ 0, 1, 2, 3 \\}$이고 $E(P_3) = \\{ \\{ 0, 1 \\}, \\{ 1, 2 \\}, \\{ 2, 3 \\} \\}$이다. 이때 $V(H) = \\{ 0, 1, 3 \\}$과 $E(H) = \\{ \\{ 0, 1 \\} \\}$을 만족하는 $P_3$의 서브그래프 $H$를 가져오자.
