## Definition 1
**(i)** A ***graph*** is a pair $G = (V, E)$ where $V$ is a finite set whose elements are called ***vertices*** and $E \subseteq \binom{V}{2}$ is a set whose elements are called ***edges***. We often write $V(G)$ and $E(G)$ for the vertex set and edge set of a given graph $G$, respectively. 

**(ii)** If $\\{ a, b \\} \in E(G)$, then we say that $a$ and $b$ are ***adjacent*** or neighbors, denoted by $a \sim b$. (Note that we only consider that a graph is an irreflexive and symmetric relation.)

---

## Definition 2
**(i)** The ***complete graph*** (or ***clique***) $K_n$ on $n$ vertices is defined by 

$$V(K_n) = [n] \quad \text{and} \quad E(K_n) = \binom{V(K_n)}{2}.$$

![alt text](assets/img/completegraph.png)

**(ii)** The ***$n$-cycle*** $C_n$ on $n$ vertices is defined by 

$$V(C_n) = [n] \quad \text{and} \quad E(C_n) = \{\{ 1, 2 \}, \{ 2, 3 \}, ..., \{ n-1, n \}, \{ n, 1 \} \}.$$

![alt text](assets/img/ncycle.png)

**(iii)** The ***$n$-path*** $P_n$ on $n$ vertices is defined by 

$$V(P_n) = \{ 0 \} \cup [n] \quad \text{and} \quad E(P_n) = \{\{ i-1, i \} \mid i \in [n] \}.$$

![alt text](assets/img/npath.png)

**(iv)** The ***complete bipartite graph*** $K_{m, n}$ on the sets $M$ and $N$ with $\vert M \vert = m$ and $\vert N \vert = n$ is defined by 

$$V(K_{m, n}) = M \cup N \quad \text{and} \quad E(K_{m, n}) = \{\{ x, y \} \mid x \in M, y \in N \}.$$

![alt text](assets/img/completebipartitegraph.png)

**(v)** A graph $G$ is ***bipartite*** if $V(G)$ has a partition $V(G) = A \cup B$ such that all edges go from $A$ to $B$.

---

## Remark
The number of graphs on $n$ vertices is $2^{\binom{n}{2}}$.
