## Definition
**(i)** A function $f$ is said to be ***analytic*** on an open set $S$ if it has a derivative everywhere on $S$. 

**(ii)** A function $f$ is said to be ***analytic*** at a point $z_0$ if it is analytic on some open set containing $z_0$.

**(iii)** A function $f$ is said to be ***entire*** if it is analytic on $\mathbb{C}$.
