## Definition
**(i)** A binary operation $\ast$ on a set $S$ is said to be ***associative*** if 

$$(a \ast b) \ast c = a \ast (b \ast c), \forall a, b, c \in S$$

**(ii)** A ***semigroup*** $(S, \ast)$ is a binary algebraic structure such that $\ast$ is associative.
