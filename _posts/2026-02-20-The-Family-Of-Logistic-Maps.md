## Logistic Map
- For $a \ge 0$, we call $g_a(x) = ax(1-x)$ the logistic map. 

---

### Properties
- If $1 < a \le 4$, then $p = \frac{a-1}{a}$ is a fixed point of the logistic map $g_a: [0, 1] \to [0, 1]$.
- If $1 < a < 3$, then $p = \frac{a-1}{a}$ is a sink, and if $a \ge 3$, then it is a source.
